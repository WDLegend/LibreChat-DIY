# OpenCode Workspace

这是一个纯 `vibe-coding` 项目。

日常使用LLM web端学习新东西，总感觉不够方便，市面上的一些产品也无法满足个人的一些需求，目前软件开发的成本也很低，打算自己用AI开发一个出来。

## 已添加内容

这段时间完成的核心内容包括：

- 为LibreChat右侧添加了web版obsidian，可以方便的记录和查看笔记。
- 添加了学习模式，开启后根据要学习的内容生成前置知识树状图，可以自己勾选已经会的部分。对快速学习新概念和进入新领域帮助很大，不用一条一条再问AI了。
- 知识树可选择性学习，可以自行切换branch。
- 学习结束后可以AI自动总结学习内容（demo）

更详细的实现记录都在 `project/` 目录里。

## project/

`project/` 用来放项目级文档，主要是：

- 已完成功能总结
- 架构理解与实现笔记
- 某一轮开发的验收记录
- 问题排查与行为说明
- 后续可继续推进的方向

如果想快速了解“这个工作区到底改了什么”，直接从 `project/` 开始看就够了。

## skills/

`skills/` 用来放辅助性的 skill 文档。

当前这里只有：

- `skills/install-skill.md`

它更偏向工作流/安装说明，不是业务功能文档。
